nohup代表后台运行  

```vim
nohup java -classpath $CLASS_PATH ${jarpath} &
```

查看磁盘使用情况  

```vim
du -h --max-depth=1
```

解压  

```vim
tar -xzf kafka_2.x.x.x.tgz
```

马上使/etc/profile文件生效  

```vim
source /etc/profile
```
